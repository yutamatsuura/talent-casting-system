# VR データファイル

## 概要
タレントキャスティングシステムで使用するVR（Visual Rating）データファイル群です。
各ファイルには、タレントの人気度と各種イメージスコア（E～K列）のデータが含まれています。

## データ構造

### ファイル内列構成
- **A列**: 順位（1, 2, 3...）
- **B列**: タレント名
- **C列**: 人気度スコア
- **D列**: 知名度スコア
- **E列**: おもしろい
- **F列**: 清潔感がある
- **G列**: 個性的な
- **H列**: 信頼できる
- **I列**: かわいい
- **J列**: カッコいい
- **K列**: 大人の魅力がある

### データ処理時の注意点
- ヘッダー行は4行目から実際のデータが開始
- エンコーディング：Shift_JIS
- 1タレントにつき最大8ターゲット層でのデータを持つ
